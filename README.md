## Getting started

* Dependency
  - Babel 
  - Eslint
  - SWR
  - google-spreadsheet
  
* To install the dependencies.
```bash
  $ npm install
  ```
  
* To run the project.
```bash
  $ npm run start
  ```







## Features

The main features of the application are:
 - Fill in the form with customer data
 - Give a note to the service
 - Show the property owner the data in a spreadsheet

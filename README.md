

## PalpiteBox
Model tool to stimulate sales, giving 10% discount to each opinion on the products.


## Technology 

Here are the technologies used in this project.

* Next JS
* React JS
* TailwindCSS
* Google-SpreadSheet

## Services Used

* Github
* Vercel
* Google Cloud API



## Getting started

* Dependency
  - Babel 
  - Eslint
  - SWR
  - google-spreadsheet
  
* To install the dependencies.
```bash
  $ npm install
  ```
  
* To run the project.
```bash
  $ npm run start
  ```
